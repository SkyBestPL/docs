## Overview

Habits Tracker provides a clean, intuitive interface for tracking daily habits with visual progress indicators, analytics, and sharing capabilities. The application is designed to be lightweight, fast, and accessible.

## Key features

### Core functionality
- **CRUD operations** - Add, edit, and delete habits with validation
- **Weekly tracking grid** - Visual 7-day view with toggle functionality
- **Data persistence** - Supabase backend with automatic saving
- **Analytics** - Charts and statistics powered by Chart.js
- **Sharing** - Generate read-only links with data encoded in URL

### User experience
- **Streak tracking** - Monitor your longest consecutive days
- **Categories** - 6 color-coded categories with dedicated icons
- **Dark/light mode** - Automatic system detection with manual toggle
- **Responsive design** - Works seamlessly from mobile to desktop
- **Accessibility** - Full keyboard navigation and screen reader support

## Technology stack

- **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3
- **Backend/Database**: Supabase (PostgreSQL)
- **Visualization**: Chart.js 4.4.0
- **Build tool**: Vite 5.4
- **Fonts**: Inter (Google Fonts)

## Use cases

Habits Tracker is perfect for:
- Building and maintaining daily routines
- Tracking fitness and wellness goals
- Learning new skills consistently
- Improving productivity habits
- Monitoring health-related activities
